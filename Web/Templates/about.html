Acerca de este Proyecto

Una aplicación educativa desarrollada para aprender programación y desarrollo web usando Python y Flask.

Sobre la Página

Este proyecto tiene como objetivo enseñar la creación de aplicaciones web interactivas, permitiendo gestionar usuarios y mostrar contenido dinámico. Ideal para estudiantes y desarrolladores que quieren aprender paso a paso cómo construir aplicaciones modernas.

Tecnologías Utilizadas

Python

Lenguaje principal para el backend y lógica de la aplicación.

Flask

Framework de Python utilizado para manejar rutas, plantillas y la interacción con la base de datos.

SQLite

Base de datos ligera y fácil de usar para almacenar usuarios y contenido de manera local.

Bootstrap

Librería CSS para dar estilo moderno y responsivo a la página web.

HTML / CSS

Lenguajes de marcado y estilos que forman la estructura y diseño visual de la aplicación.
